# Livestock Health Monitoring System

A Machine Learning-based web application that helps detect livestock diseases using health-related data. The project predicts possible diseases based on user input and provides quick insights for farmers and veterinarians.

## Features

- Disease prediction using Machine Learning
- Simple web interface built with Flask
- Trained model for livestock disease detection
- Dataset included for training and testing
- Easy to run locally using Python

## Tech Stack

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- Jupyter Notebook

## Project Structure

```
├── app.py
├── Livestock_Health_Monitoring_System.ipynb
├── animal_disease_dataset.xlsx
├── livestock_health_model_artifacts.zip
├── requirements.txt
├── Dockerfile
└── README.md
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Livestock-Health-Monitoring-System.git
cd Livestock-Health-Monitoring-System
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python app.py
```

4. Open your browser and visit

```
http://localhost:5000
```

## Dataset

The project uses the `animal_disease_dataset.xlsx` dataset for training and prediction.
